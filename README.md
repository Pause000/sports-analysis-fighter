## 프로젝트 소개
- 프로젝트 명 : 사용자 성향 기반 스포츠 팀 추천 챗봇
- 목적
  - 내가 응원하는 팀이 있다면 내가 응원하는 팀과 비슷한 성향을 가진 다른 종목의 팀을 추천
  - 스포츠 팬이 아니여도 쉽게 응원 팀을 선택할 수 있도록 도움
- 형태
  - 스포츠 충계 사이트에 탑재되는 대화형 챗봇 서비스
 
## 주요 기능
- 1️⃣ 사용자 성향 분석
  - 간단한 질문을 통해 사용자의 성향 수집 : 역사 , 스타선수 선호 , 전통 있는 팀 , 감성 , 승부욕등 ...
  - 선택지 기반 질문 (초보자도 쉽게 응답가능)

- 2️⃣ 스포츠 팀 데이터 관리
  - 전력
  - 역사 , 전통
  - 스타성
  - 팬 문화

- 3️⃣ 웹 챗봇 UI
  - 질문 -> 응답 -> 추천 결과 흐름
  - HTML / CSS 기반의 간단한 챗봇 인터페이스
  - 스포츠 중계 사이트 삽입을 고려한 구조

- 4️⃣ 팀 추천 로직
  - 사용자 성향 점수와 팀 속성 점수를 비교
  - 가중치 기반 점수 계산
  - 가장 적합한 팀을 추천
