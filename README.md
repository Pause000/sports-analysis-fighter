# sports-analysis-fighter

## 프로젝트 개요
- 프로젝트 명 : 사용자 성향 기반 스포츠 팀 추천 챗봇
- 목적
  - 내가 응원하는 팀이 있다면 내가 응원하는 팀과 비슷한 성향을 가진 다른 종목의 팀을 추천
  - 스포츠 팬이 아니여도 쉽게 응원 팀을 선택할 수 있도록 도움
- 형태
  - 스포츠 충계 사이트에 탑재되는 대화형 챗봇 서비스
 
## 주요 기능
- 사용자 성향 분석
  - 간단한 질문을 통해 사용자의 성향 수집 : 역사 , 스타선수 선호 , 감성
  - 선택지 기반 질문 (초보자도 쉽게 응답가능)

- 스포츠 팀 데이터 관
